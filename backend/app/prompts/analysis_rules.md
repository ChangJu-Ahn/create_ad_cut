# 분석 규칙 — Product Descriptor 작성

출력은 **상품 외형 정의 (product descriptor)** 입니다.
인물·맥락·동작은 descriptor 어휘 범위 밖이며, 다음 9개 축만 기술합니다.

## 1. 카테고리·형태·실루엣
- 카테고리 (예: 셔츠, 블레이저, 토트백, 스니커즈)
- 실루엣 (슬림/레귤러/오버사이즈, 박시/A라인 등), 길이, 어깨선, 본체 비율

## 2. 색상
- 메인 컬러 + 보조 컬러를 분리해 명시
- 톤 (밝기), 채도, 광택 여부
- 색상 순서가 있는 경우 (예: 단추 띠, 스트라이프) 등장 순서대로 기록

## 3. 재질·텍스처
- 직조 방식 (니트, 데님, 풀그레인 가죽 등), 두께감, 광택 정도
- 사진에서 보이는 단서만 사용

## 4. 마감부
- 밑단, 커프스, 카라, 플래킷, 솔기 처리, 배색 라인
- 가방류: 스트랩 길이/너비, 거싯 두께, 모서리 마감
- 신발류: 토캡 형태, 아웃솔 패턴, 힐 컵 마감

## 5. 부속 요소
- 단추 (개수·색·재질), 지퍼, 포켓 (개수·위치·형태)
- 패치, 자수, 로고 (위치·크기·색), 버클, 아일렛

## 6. 좌우 비대칭
- 이미지 기준 왼쪽 / 오른쪽으로 분리해 기술
- 두 쪽이 다르게 보이는 경우 일반화하지 말 것

## 7. 가려진 부위
- 손, 팔, 다른 옷 등에 가려진 부분은 "보이는 범위에서는" 표현으로 표시
- 보이지 않는 부분을 임의로 그려 넣지 말 것

## 8. 어휘 범위 (출력 형식)
Product descriptor 는 **상품 자체의 정적 속성** 만 담습니다.

사용할 어휘 형태:
- 명사구: "라운드 토캡", "무광 골드 메탈 버클", "포멀 노치드 라펠"
- 형용사구: "매트한 표면", "미세 그레인 패턴", "두꺼운 풀그레인"
- 정적 묘사: "본체 정면에 가로 약 8cm 의 브랜드 로고가 압인되어 있다"

출력 본문은 상품의 카테고리/형태/색/재질/마감/부속만 다룹니다.
인물·동작·맥락 (모델, 손, 발, 포즈, 장소, 이벤트, 착용/소지 동사) 은
 descriptor 의 대상이 아니므로 자연스럽게 작성에서 제외됩니다.

## 9. 정체성 보존 문장
다음 표현 중 적절한 것을 본문에 자연스럽게 포함:
- "정확히 보존"
- "이미지 기준 좌우를 유지"
- "일반화하지 말 것"
- 마지막 줄 (필수): "이미지 기준 왼쪽/오른쪽을 유지하고 좌우 반전하지 말 것."

## 입력 자료 안내
- 첫 번째 이미지는 원본 풀 이미지입니다.
- 그 다음 이미지들은 자동 생성된 detail crop 입니다 (좌/우 절반, 상/하 절반,
  밑단 띠 영역 등). detail crop 은 full image 와 교차 확인용으로만 사용하고,
  결과 프롬프트에는 "crop" 이라는 단어를 노출하지 마세요.
